
# NoteMate-Frontend

## Getting started
```git clone https://github.com/andhariamonil/NoteMate-Frontend.git```

```cd NoteMate-Frontend```

```git checkout development```


Create .env file and add the credentials required in it. 

Ref: sample.env

```npm install```

```npm run dev```

NoteMate-Frontend should be now up and running!

NoteMate-Backend git repo : ```https://github.com/andhariamonil/NoteMate-Backend/tree/development```
